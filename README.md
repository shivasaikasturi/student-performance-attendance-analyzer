## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on a student performance dataset to identify the academic and engagement factors that influence student success.

Using **Python, Pandas, NumPy, Matplotlib, and Seaborn**, the project analyzes GPA, CGPA, attendance, classroom participation, learning resource utilization, and discussion activities through statistical analysis and data visualization.

---

## 🎯 Objectives

- Analyze student academic performance.
- Explore attendance patterns.
- Study classroom participation.
- Analyze learning resource utilization.
- Perform data cleaning and preprocessing.
- Generate insightful visualizations.
- Draw data-driven conclusions.

## 📈 Key Insights

- Students with higher classroom participation generally achieved higher GPA.
- Frequent learning resource usage positively influenced academic performance.
- Good attendance was associated with stronger academic outcomes.
- Student engagement demonstrated a positive relationship with GPA and CGPA.
- Data visualization revealed meaningful academic performance trends.
